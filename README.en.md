# zsz-qwq.github.io

This is a personal website project hosted on GitHub Pages.

## Introduction

This project is designed to host a personal blog or resume website, and can be automatically deployed via Gitee Pages.

## Usage

1. Clone this repository
2. Modify `index.html` or other page files to update the website content
3. Push to the remote repository; Gitee Pages will automatically build and deploy the website
